# Tennessee Stranded Talent Interactive Explorer

A static, single-page dashboard that visualizes the Burning Glass Institute's
analysis of career-stalled and stranded workers in Tennessee — by metro area,
industry sector, and occupation, with next-step occupation pathways and skill
gaps.

**Live site:** https://burning-glass-institute.github.io/tn-stranded-talent-dashboard/

## About the data

Everything the app displays is a **precomputed aggregate**. The `viewstore/`
directory holds a small set of JSON files — geography/sector summaries, pathway
lists, and occupation reference attributes — that are exactly the figures rendered
on screen. No individual-level records are included or downloadable; the app does
no client-side aggregation. Figures are generated by the analysis pipeline and
published here as a read-only snapshot.

## Development

```bash
npm install
npm run dev        # prebuild sync copies viewstore/ -> public/data/, then Vite serves at :3000
npm run build      # -> dist/ (postbuild bundle guard enforces no microdata in the JS bundle)
npm run preview
```

The dashboard is a Vite + React app. `npm run build` runs a bundle guard
(`scripts/check_bundle.mjs`) that fails if raw microdata column names or
spreadsheets ever appear in the compiled JS.

## Deployment

Pushes to `main` build and publish to GitHub Pages via
`.github/workflows/deploy-pages.yml`. The project is served under the
`/tn-stranded-talent-dashboard/` base path (set by `VITE_BASE` in the workflow).

## Provenance

This repository is a published mirror. Its contents are exported from the
BGI analysis repository by an automated allowlist-based script; only the app and
the view-store aggregates are copied here.
